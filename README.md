# Kalang
A toy compiler front-end

# Features

* static type safe
* jvm compatible
* auto type
* java source output

# How to build

clone the source,and run

    >gradle installDist

# Usage

add the path`build/install/kalang/bin` to your `PATH` environment variable,and run

    >kalang SOURCE_DIR OUTPUT_DIR
